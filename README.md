# AI Ambassadors

The official website for **AI Ambassadors**, a student-led team based in Thessaloniki, Greece, exploring how artificial intelligence works and how it can be used ethically in our everyday lives.

🔗 **Live site:** https://az0u.github.io/AI-Ambassadors/

## About

AI Ambassadors is a student-run initiative founded in 2024. Through workshops, collaborations, and presentations, the team makes AI more understandable for students while raising awareness about its ethical implications and encouraging responsible use.

This repository contains the source code for the team's public website, which covers:

- **About:** who the team is and what they do
- **Events:** outreach events and workshops hosted so far
- **Educational Content:** articles and videos created by team members
- **Team:** the people behind the initiative

## Tech Stack

This is a static website, which any browser can run directly, built with:

- **HTML5**
- **CSS3** 
- **JavaScript**
